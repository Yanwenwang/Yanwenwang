### Scramble Word

- Called wordnik API with $http to pick a random word to scramble
- Coded algorithm to scramble word for the player to guess
- Listened for keydown event to allow the player to use keyboard to type guesses 
- Added support for backspace using event.preventDefault to stop the browser from going to previous page which allows the player to remove a guessed letter
- Included scoring system and countdown timer, using $interval
- Animated word on correct/incorrect guesses using Animate.css and $timeout
- Surfaced modal when countdown timer runs out using Angular UI Bootstrap
- Made game responsive using Bootstrap and Media Queries
